# /campagne

Action : Lancer une campagne end-to-end
Workflow :
1. Stratège → brief (validation utilisateur)
2. Créateur → contenu (validation utilisateur)
3. Designer → prompts images (validation utilisateur)
4. Présentateur → deck (validation utilisateur)
5. (optionnel) Analyste → rapport à J+30
Description : Lance la chaîne complète des 5 agents séquentiellement avec validation à chaque étape critique.
